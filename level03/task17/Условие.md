
### Форматирование и сортировка списка студентов

У вас есть таблица `students`, которая содержит данные о студентах.

Ваша задача:

1. Объединить столбцы `first_name` и `last_name` в один столбец `full_name` в формате "Имя Фамилия".

2. Извлечь уникальные записи (чтобы не повторялись студенты с одинаковыми именами и фамилиями).

3. Отсортировать студентов по году поступления (`enrollment_year`) в порядке возрастания.

Результат запроса должен содержать столбцы `full_name` и `enrollment_year`.

#### Требования:
1. Запрос должен использовать функцию `CONCAT()` или оператор `||` для объединения значений столбцов `first_name` и `last_name` в новый столбец `full_name` в формате "Имя Фамилия".
2. Запрос должен использовать ключевое слово `DISTINCT` для исключения дублирующихся записей с одинаковыми значениями в столбцах `full_name` и `enrollment_year`.
3. Запрос должен использовать оператор `ORDER BY` для сортировки данных по столбцу `enrollment_year` в порядке возрастания.
4. Результат запроса должен содержать только столбцы `full_name` и `enrollment_year`.
