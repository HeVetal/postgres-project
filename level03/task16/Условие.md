У вас есть таблица `orders` с колонками: `order_id` (целое число), `customer_id` (целое число), `order_date` (дата), `total` (целое число).

Напишите SQL-запрос, который возвращает все заказы, отсортированные:

1. Сначала по дате заказа в порядке возрастания (`ASC`);

2. При одинаковой дате — по идентификатору клиента в порядке убывания (`DESC`).

Результат должен упорядочиться так, чтобы заказы с одинаковыми датами сначала выводились для клиента с наибольшим `customer_id`.

#### Требования:
1. Запрос должен обращаться к таблице `orders`, содержащей указанные столбцы: `order_id`, `customer_id`, `order_date`, `total`.
2. Результат должен быть отсортирован по столбцу `order_date` в порядке возрастания (`ASC`).
3. Для строк с одинаковым значением `order_date`, результат должен быть отсортирован по столбцу `customer_id` в порядке убывания (`DESC`).
4. Запрос должен возвращать все столбцы таблицы `orders` в результате.
