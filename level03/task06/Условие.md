
### Извлечение года и месяца из текущей даты

Напишите SQL-запрос, который использует функцию `DATE_PART()` для извлечения текущего года и месяца. Запрос должен вернуть два столбца: `current_year` и `current_month`.

#### Требования:
1. Запрос должен использовать функцию `DATE_PART()` для извлечения года и месяца из текущей даты.
2. Функция `DATE_PART()` должна быть применена с параметром `'year'` для извлечения текущего года.
3. Функция `DATE_PART()` должна быть применена с параметром `'month'` для извлечения текущего месяца.
4. Результат запроса должен содержать два столбца с именами `current_year` и `current_month`.
