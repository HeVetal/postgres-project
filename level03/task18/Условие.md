
### Уникальные заказы с форматированием клиентов

У вас есть таблица `orders`, которая содержит данные о заказах.

Ваша задача:

1. Преобразовать дату заказа (`order_date`) в формат `DD.MM.YYYY` и записать в `formatted_order_date`.

2. Отсортировать записи по `formatted_order_date` в порядке убывания.

3. Отобразите столбцы `customer_name`, `formatted_order_date` и `total_amount`.

4. Используйте DISTINCT для устранения дубликатов.

#### Требования:
1. Запрос должен преобразовать значение в столбце `order_date` в формат `DD.MM.YYYY` и сохранить результат в столбце `formatted_order_date`.
2. Результат должен быть отсортирован по столбцу `formatted_order_date` в порядке убывания.
3. Используйте DISTINCT для устранения дубликатов.
4. Результат должен содержать только три столбца: `customer_name`, `formatted_order_date` и `total_amount`.
