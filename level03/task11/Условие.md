
### Определение длины строки

В таблице `users` содержатся данные о пользователях: имена и email-адреса.

Напишите SQL-запрос, который возвращает имена пользователей и длину их имен в отдельном столбце с названием `name_length`.

#### Требования:
1. Запрос должен использовать функцию LENGTH() для определения длины строки в столбце name.
2. Запрос должен возвращать два столбца: name и name_length, где name содержит имена пользователей, а name_length — длину их имен.
3. Запрос не должен изменять данные в таблице users, а только извлекать их.
4. Столбец с длиной имени должен быть явно назван как name_length в результирующем запросе.
5. Функция LENGTH() должна учитывать пробелы в имени при подсчете длины строки.
