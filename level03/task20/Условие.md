
### Сортировка по нескольким столбцам с разными направлениями

У вас есть таблица `products` с колонками `id` (целое число), `category` (текст) и `price` (число с плавающей точкой).

Напишите запрос, который сортирует товары сначала по категории (по возрастанию), а внутри каждой категории — по цене (по убыванию).

#### Требования:
1. Запрос должен сортировать данные сначала по столбцу `category` в порядке возрастания, а затем по столбцу `price` в порядке убывания.
2. Для каждой колонки сортировки должно быть явно указано направление: `ASC` для `category` и `DESC` для `price`.
3. Запрос должен возвращать все исходные столбцы таблицы `products` (`id`, `category`, `price`) без изменений.
