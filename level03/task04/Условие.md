
### Стандартизация email-адресов

В таблице `customers` имеется столбец `email`.

Напишите SQL-запрос, который выведет значения столбца `email` в нижнем регистре в столбце с названием `email_lower`.

Кроме того, добавьте ещё один столбец с названием `email_upper`, где каждый email будет в верхнем регистре.

Колонки результата: email, email_lower, email_upper

#### Требования:
1. Запрос должен использовать функцию `LOWER()` для преобразования значений столбца `email` в нижний регистр.
2. Запрос должен использовать функцию `UPPER()` для преобразования значений столбца `email` в верхний регистр.
3. Результат запроса должен содержать два дополнительных столбца: `email_lower` и `email_upper`, которые будут содержать преобразованные значения.
