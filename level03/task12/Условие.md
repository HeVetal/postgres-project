
### Извлечение части строки

В таблице `users` содержатся данные о пользователях: имена и email-адреса.

Напишите SQL-запрос, который извлекает домен из email-адресов (часть адреса после символа `@`). Результат должен содержать два столбца: `email` и `domain`.

#### Требования:
1. Запрос должен использовать функцию `SUBSTRING()` для извлечения части строки после символа `@` из столбца `email`.
2. Для определения начала извлекаемой части строки необходимо использовать символ `@` как точку отсчёта в функции `SUBSTRING()`.
3. Результат запроса должен содержать два столбца: `email` (оригинальный email-адрес) и `domain` (извлечённая часть строки после символа `@`).
4. Запрос должен извлекать данные из таблицы `users`, содержащей столбцы `user_id`, `name` и `email`.
