
### Объединение столбцов и форматирование строк

В таблице `students` хранятся данные о студентах.

Напишите запрос, который:

1. Объединяет имя и фамилию студента в один столбец под названием `full_name`.

2. Выводит этот столбец вместе с датой рождения, отформатированной в виде `DD-MM-YYYY` под названием `formatted_birth_date`.

#### Требования:
1. Запрос должен использовать оператор `||` для объединения строк, чтобы объединить `first_name` и `last_name` в один столбец `full_name`.
2. Запрос должен использовать функцию форматирования даты, например, `TO_CHAR()`, для преобразования значения `birth_date` в формат `DD-MM-YYYY`.
3. Результирующий столбец с объединённым именем и фамилией должен быть назван `full_name`, а столбец с отформатированной датой рождения — `formatted_birth_date`.
4. Запрос должен извлекать только два столбца: `full_name` и `formatted_birth_date`, без дополнительных данных.
