
### Сортировка студентов по возрасту и имени

У вас есть таблица `students` с колонками: `id` (целое число), `first_name` (текст), `last_name` (текст), `age` (целое число).

Напишите SQL-запрос, который возвращает всех студентов, отсортированных:

1. Сначала по возрасту в порядке убывания (`DESC`);

2. Затем по имени в порядке возрастания (`ASC`).

Результат должен быть упорядочен так, чтобы сначала шли более старшие студенты, а при одинаковом возрасте — в алфавитном порядке по имени.

#### Требования:
1. Запрос должен обращаться к таблице `students`, которая содержит колонки `id`, `first_name`, `last_name`, `age`.
2. Данные должны быть отсортированы по колонке `age` в порядке убывания (`DESC`), чтобы сначала шли более старшие студенты.
3. При одинаковом значении возраста студенты должны быть отсортированы по колонке `first_name` в порядке возрастания (`ASC`).
4. Запрос должен возвращать все колонки таблицы `students` (`id`, `first_name`, `last_name`, `age`).
