
### Объединение строк

В таблице `employees` хранятся данные о сотрудниках.

Напишите SQL-запрос, который объединяет имя (first_name) и фамилию (last_name) каждого сотрудника в строку с пробелом между ними. Результат должен содержать столбец `full_name`, а также `employee_id`.

#### Требования:
1. Запрос должен использовать `оператор ||` или функцию `CONCAT` для объединения значений столбцов `first_name` и `last_name` с добавлением пробела между ними.
2. Результат запроса должен содержать столбец `full_name`, в котором объединены имя и фамилия сотрудников.
3. Результат должен включать столбец `employee_id`, чтобы сохранить идентификатор каждого сотрудника.
4. Объединение имени и фамилии должно включать ровно один пробел между ними.
5. Запрос должен быть направлен на данные из таблицы `employees`.
