
### Уникальные города клиентов

В таблице `customers` есть столбец `city`, который хранит название городов, где проживают клиенты.

Напишите SQL-запрос, который выводит только уникальные названия городов в столбце с названием `unique_city`.

#### Требования:
1. Запрос должен использовать оператор `DISTINCT` для извлечения уникальных значений из столбца `city`.
2. Результат должен содержать столбец с названием `unique_city`, что требует использования алиаса (псевдонима) для столбца.
3. Запрос должен обращаться к таблице `customers` и извлекать данные из столбца `city`.
4. В результирующем наборе данных не должно быть повторяющихся значений названий городов.
