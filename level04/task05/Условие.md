
### Суммирование всех элементов столбца

В таблице `orders` содержатся данные о заказах.

Напишите запрос, который рассчитает общую сумму всех заказов и выведет это значение в столбце с названием `total_order_amount`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `SUM()` для расчета общей суммы значений в столбце `order_amount`.
2. Результат запроса должен быть представлен в виде столбца с именем `total_order_amount`.
3. Запрос должен извлекать данные из таблицы `orders`, содержащей столбцы `order_id` и `order_amount`.
