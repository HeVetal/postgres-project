
### Фильтрация студентов по факультетам с количеством больше двух

У вас есть таблица `students` с данными студентов.

Напишите SQL-запрос, который группирует студентов по факультетам и возвращает названия этих факультетов, где количество студентов больше двух.

#### Требования:
1. Запрос должен использовать функцию `COUNT()` для подсчёта количества студентов в каждой группе.
2. Запрос должен группировать строки по значению столбца `faculty` с использованием оператора `GROUP BY`.
3. Запрос должен включать условие фильтрации с использованием ключевого слова `HAVING`, чтобы выбрать только те группы, где количество студентов больше двух.
4. Запрос должен возвращать два столбца: `faculty` и `student_count`, где `student_count` — это результат функции `COUNT()`.
