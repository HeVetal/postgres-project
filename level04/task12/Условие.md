
### Подсчёт количества продуктов по категориям

В таблице `products` содержатся данные о продуктах.

Напишите SQL-запрос, который группирует данные по категориям (`category`) и считает количество продуктов в каждой категории. Запрос должен вернуть два столбца: `category` и `product_count`.

#### Требования:
1. Запрос должен группировать строки таблицы `products` по значению столбца `category`.
2. Для подсчёта количества продуктов в каждой категории необходимо использовать агрегатную функцию `COUNT()`.
3. Результат запроса должен содержать два столбца: `category` (название категории) и `product_count` (количество продуктов в данной категории).
