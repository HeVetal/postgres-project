
### Суммирование с условием

Таблица `sales` хранит информацию о продажах.

Напишите запрос, который рассчитает общую сумму продаж (`sale_amount`) только для категории `electronics`. Выведите результат в столбце с названием `total_electronics_sales`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `SUM()` для расчета общей суммы значений столбца `sale_amount`.
2. Запрос должен включать условие фильтрации с использованием `WHERE` для выбора только тех строк, где значение в столбце `sale_category` равно `electronics`.
3. Результат запроса должен быть выведен в столбце с названием `total_electronics_sales`.
4. Запрос должен извлекать данные только из таблицы `sales`, без использования дополнительных таблиц или соединений.
