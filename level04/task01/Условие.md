
### Список товаров со скидкой

У вас есть таблица `products` с данными о товарах.

Ваша задача:

1. Вычислить новую цену со скидкой (`discounted_price`) по формуле:

`price - (price * discount_percent / 100)`.

2. Отфильтровать товары, у которых есть скидка (`discount_percent > 0`).

3. Упорядочить результат по `discounted_price` в порядке возрастания.

Результат должен содержать столбцы `product_name`, `price`, `discount_percent`, `discounted_price`.

#### Требования:
1. Необходимо вычислить новую цену со скидкой (`discounted_price`) по формуле: `price - (price * discount_percent / 100)`.
2. В запросе должны быть отфильтрованы только те товары, у которых поле `discount_percent` больше 0.
3. Результат запроса должен быть упорядочен по столбцу `discounted_price` в порядке возрастания.
4. Результат должен содержать следующие столбцы: `product_name`, `price`, `discount_percent`, `discounted_price`.
