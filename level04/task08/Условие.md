
### Расчет средней зарплаты сотрудников с условием

У вас есть таблица `employees` с данными сотрудников.

Напишите запрос, который вычислит среднюю зарплату только тех сотрудников, у которых зарплата больше 50000. Назовите результирующий столбец `average_salary`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `AVG()` для расчета среднего значения.
2. Запрос должен включать условие `WHERE salary > 50000` для отбора сотрудников с зарплатой больше 50000.
3. Функция `AVG()` должна автоматически игнорировать строки, где значение зарплаты равно `NULL`.
4. Результирующий столбец должен быть назван `average_salary` с использованием оператора `AS`.
