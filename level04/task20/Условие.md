
### Фильтрация с помощью HAVING

В таблице `orders` хранятся данные о заказах.

Напишите запрос, чтобы найти регионы (`region`), где общая сумма заказов превышает 600.

Отсортируйте результат в порядке убывания общей суммы заказов.

Выведите два столбца: `region` и `total_sales`.

#### Требования:
1. Запрос должен использовать функцию `SUM()` для вычисления общей суммы заказов (`total_amount`) по регионам.
2. Запрос должен группировать данные по столбцу `region` с использованием оператора `GROUP BY`.
3. Запрос должен использовать оператор `HAVING` для фильтрации данных, чтобы отобрать только те регионы, где общая сумма заказов превышает 600.
4. Запрос должен сортировать результат в порядке убывания общей суммы заказов (`total_sales`) с использованием оператора `ORDER BY` и ключевого слова `DESC`.
5. Запрос должен выводить только два столбца: `region` и `total_sales`.
