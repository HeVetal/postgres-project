
### Группировка клиентов по странам

В таблице `customers` содержатся данные о клиентах.

Напишите SQL-запрос, который групирует клиентов по странам и считает общее количество заказов (`orders_count`) в каждой стране. Запрос должен вернуть два столбца: `country` и `total_orders`.

#### Требования:
1. Запрос должен использовать данные из таблицы `customers`, содержащей информацию о клиентах, включая их страну и количество заказов.
2. Запрос должен группировать данные по столбцу `country`, чтобы объединить всех клиентов из одной страны.
3. Для вычисления общего количества заказов по каждой стране необходимо использовать агрегатную функцию `SUM()` на столбце `orders_count`.
4. Запрос должен возвращать два столбца: `country` (название страны) и `total_orders` (общее количество заказов в этой стране).
