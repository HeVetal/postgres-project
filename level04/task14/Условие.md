
### Фильтрация продуктов по общей сумме продаж

У вас есть таблица `sales` с данными о продажах.

Напишите SQL-запрос, который группирует данные по идентификатору продукта (`product_id`) и возвращает только те продукты, суммарные продажи которых превышают 500.

#### Требования:
1. Запрос должен использовать функцию `SUM()` для вычисления общей суммы продаж по каждому идентификатору продукта (`product_id`).
2. Запрос должен группировать данные по столбцу `product_id` с использованием оператора `GROUP BY`.
3. Запрос должен содержать условие `HAVING` для фильтрации групп, где суммарные продажи превышают 500.
4. Запрос должен возвращать только два столбца — `product_id` и `total_sales`, где `total_sales` представляет собой сумму продаж для каждой группы.
