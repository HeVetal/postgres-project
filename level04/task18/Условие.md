
### Расчёт площадей круга

В таблице `circles` есть столбец `radius`, который хранит радиусы кругов. Напишите SQL-запрос, который вычисляет площадь каждого круга по формуле:

`π * r^2`, где `r` — радиус круга. Используйте значение `π = 3.14159` и функцию `POWER()`.

Запрос должен вернуть два столбца:

1. `radius` — радиус круга.

2. `area` — рассчитанная площадь круга.

#### Требования:
1. Запрос должен использовать функцию `POWER()` для возведения радиуса в квадрат.
2. В запросе необходимо использовать значение `π = 3.14159`.
3. Запрос должен корректно вычислять площадь круга по формуле `π * r^2`.
4. Запрос должен возвращать два столбца: `radius` (исходный радиус) и `area` (рассчитанная площадь круга).
5. Запрос должен обеспечивать точность вычислений, чтобы результат соответствовал заданным примерам.
