
### Нахождение минимального и максимального количества

В таблице `products` есть столбец `quantity`, который хранит количество товаров.

Напишите SQL-запрос, который вернет минимальное и максимальное количество товаров.

Назовите столбцы в выводе `min_quantity` и `max_quantity`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `MIN()` для нахождения минимального значения в столбце `quantity`.
2. Запрос должен использовать агрегатную функцию `MAX()` для нахождения максимального значения в столбце `quantity`.
3. Результат запроса должен содержать два столбца с именами `min_quantity` и `max_quantity`.
