
### Проблемы с неагрегированными столбцами

В таблице `sales` содержаться данные о продажах.

Напишите запрос, чтобы найти общую сумму продаж по каждому отделу (`department`).

Вывести нужно только два столбца: `department` и `total_sales`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `SUM()` для вычисления общей суммы продаж (`sale_amount`) по каждому отделу (`department`).
2. Запрос должен содержать оператор `GROUP BY`, чтобы сгруппировать данные по столбцу `department`.
3. Результат запроса должен содержать только два столбца: `department` и `total_sales`.
