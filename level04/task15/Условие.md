
### Найти факультеты с числом студентов старше 18 больше 5

В таблице `students` напишите SQL-запрос, который:

1. Сначала выбирает только строки, где возраст студентов больше 18.

2. Сгруппировывает данные по факультету (`department`).

3. Показывает факультеты с количеством студентов, удовлетворяющих условию, больше 5.

Выведите два столбца: `department` и `student_count` (общее количество студентов на этом факультете).

#### Требования:
1. Запрос должен включать условие фильтрации `WHERE age > 18` для отбора только тех студентов, чей возраст превышает 18 лет.
2. Данные должны быть сгруппированы по полю `department` с использованием оператора `GROUP BY`.
3. Для отбора факультетов с количеством студентов больше 5 должно использоваться условие `HAVING COUNT(*) > 5`.
4. Результат запроса должен содержать два столбца: `department` и `student_count`, где `student_count` представляет количество студентов, соответствующих условиям.
