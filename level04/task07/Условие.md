
### Простой расчет среднего возраста студентов

У вас есть таблица `students` с данными студентов.

Напишите запрос, который вычислит средний возраст всех студентов. Назовите результирующий столбец `average_age`.

#### Требования:
1. Запрос должен использовать агрегатную функцию `AVG()` для вычисления среднего значения по столбцу `age`.
2. Результирующий столбец должен быть назван как `average_age` с использованием алиаса.
3. Запрос должен извлекать данные из таблицы `students`, которая содержит столбцы `id`, `name` и `age`.
4. Средний возраст должен быть рассчитан на основе всех значений столбца `age` без фильтрации или исключений.
