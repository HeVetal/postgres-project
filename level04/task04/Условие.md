
### Подсчёт строк с ненулевой ценой

Используя ту же таблицу `products`, напишите SQL-запрос, который подсчитает количество продуктов, у которых указана цена (`price` не равна `NULL`).

#### Требования:
1. Запрос должен использовать агрегатную функцию `COUNT()` для подсчета строк.
2. Запрос должен учитывать только строки, где значение столбца `price` не равно `NULL`.
3. В функции `COUNT()` должен быть указан конкретный столбец (`price`), а не `COUNT(*)`.
