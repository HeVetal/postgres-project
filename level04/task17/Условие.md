
### Округление чисел до целого

Напишите SQL-запрос, который извлекает стоимость товаров из таблицы `products` (столбец `price`) и округляет её до ближайшего целого значения с помощью функции `ROUND()`.

Запрос должен вернуть два столбца:

1. `original_price` — исходная стоимость товара.

2. `rounded_price` — округлённая стоимость товара.

#### Требования:
1. Запрос должен использовать функцию ROUND() для округления значений в столбце price до ближайшего целого числа.
2. Запрос должен извлекать данные из таблицы products, обращаясь к столбцу price.
3. Результат запроса должен содержать два столбца: original_price (исходное значение из столбца price) и rounded_price (округлённое значение).
4. Столбец original_price должен отображать исходное значение из столбца price без изменений.
