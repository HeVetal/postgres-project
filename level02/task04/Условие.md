
### Выбор конкретных столбцов в SQL

В таблице `students` есть столбцы: `id`, `name`, `age`, `course`.

Напишите SQL-запрос, который выбирает только следующие столбцы: `name` и `course`.

Ожидаемый результат: запрос должен отобразить список всех студентов с указанием их имени и курса обучения.

#### Требования:
1. Запрос должен использовать команду `SELECT` для извлечения данных из таблицы.
2. В запросе должны быть указаны только столбцы `name` и `course` из таблицы `students`.
3. Запрос должен явно указывать таблицу `students` как источник данных.
