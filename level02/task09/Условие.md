
### Использование оператора `AND` для фильтрации данных

У вас есть таблица `employees` с информацией про сотрудников.

Напишите запрос, который выберет сотрудников из IT-департамента с зарплатой больше 70000.

Выведите только столбцы `name`, `department` и `salary`.

#### Требования:
1. Запрос должен использовать оператор `AND` для комбинирования двух условий: принадлежность сотрудника к IT-департаменту и зарплата больше 70000.
2. Запрос должен фильтровать строки, где значение в столбце `department` равно 'IT'.
3. Запрос должен фильтровать строки, где значение в столбце `salary` больше 70000.
4. Запрос должен возвращать только столбцы `name`, `department` и `salary`.
