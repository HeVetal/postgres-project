
### Формирование нового столбца с использованием данных из других столбцов

У вас есть таблица `products` с информацией про товары.

Напишите SELECT-запрос, который:

1. Выводит два столбца: `Описание` и `price`.

2. Столбец `Описание` объединяет значения столбцов `product_name` и `category` через дефис (`-`).

Пример вычисления содержимого столбца `Описание`:

Для первой строки результат будет: `Телевизор - Электроника`.

#### Требования:
1. Запрос должен основываться на данных из таблицы `products`.
2. Запрос должен создавать новый столбец `Описание`, который объединяет значения столбцов `product_name` и `category` через дефис (`-`).
3. Результат запроса должен содержать только два столбца: `Описание` и `price`.
