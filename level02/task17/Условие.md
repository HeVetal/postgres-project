
### Фильтр и сортировка товаров

У вас есть таблица `products`, которая содержит данные о товарах.

Напишите SQL-запрос, который выберет название и цену всех товаров из категории "Appliances", упорядочив их по цене по убыванию.

#### Требования:
1. Запрос должен извлекать только два столбца из таблицы products: product_name и price.
2. Необходимо отфильтровать данные по значению category = 'Appliances' с использованием оператора WHERE.
3. Результаты должны быть отсортированы по столбцу price в порядке убывания с помощью конструкции ORDER BY price DESC.
