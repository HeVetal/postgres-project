
### Постраничный вывод данных

В таблице `students` хранятся данные о студентах.

Напишите SQL-запрос, который возвращает студентов, начиная со второго и ограничивает результат двумя строками.

#### Требования:
1. Запрос должен использовать оператор OFFSET для пропуска первой строки в таблице `students`.
2. Запрос должен использовать оператор LIMIT для ограничения результата двумя строками.
3. Запрос должен возвращать только указанные в примере столбцы: `student_id`, `name`, `age`.
4. Запрос должен быть построен для работы с таблицей `students`, содержащей данные о студентах.
