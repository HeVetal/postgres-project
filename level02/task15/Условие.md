
### Пропуск строк с использованием `OFFSET`

В таблице `products` хранятся данные о продуктах:

Напишите SQL-запрос, который возвращает все товары, начиная с третьего по порядку.

#### Требования:
1. Запрос должен включать оператор `OFFSET` для пропуска первых двух строк в таблице.
2. Должен быть явно задан порядок строк, чтобы определить, какие строки считаются первыми, вторыми и так далее.
3. Запрос должен возвращать все столбцы из таблицы `products` — `product_id`, `name`, `price`.
4. Запрос должен выполняться на таблице `products`.
