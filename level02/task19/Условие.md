
### Исправление синтаксической ошибки

Вам нужно исправить SQL-запрос, чтобы он корректно выполнялся.

После исправления запрос должен возвращать идентификатор студента (`id`), полное имя (`full_name`) и возраст (`age`) для всех студентов 10-го класса. Названия столбцов в результате должны быть такими же, как их названия в таблице.

#### Требования:
1. Запрос должен быть исправлен таким образом, чтобы он соответствовал синтаксису SQL и выполнялся без ошибок.
2. В списке выбираемых столбцов (`id`, `full_name`, `age`) должны быть использованы запятые для их разделения.
3. Запрос должен обращаться к таблице `students`, как указано в условии.
4. Должно быть указано условие фильтрации `WHERE grade = 10`, чтобы возвращались данные только для студентов 10-го класса.
