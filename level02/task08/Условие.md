
### Поиск товаров с низкой ценой

В таблице `products` хранится информация о продуктах.

Напишите SQL-запрос, который возвращает названия (`product_name`) и цены (`price`) всех товаров, цена которых меньше 50.

#### Требования:
1. Запрос должен использовать оператор сравнения `<` для фильтрации товаров с ценой меньше 50.
2. Запрос должен выбирать только столбцы `product_name` и `price` из таблицы `products`.
3. Запрос должен обращаться к таблице `products` для получения данных.
4. Запрос должен возвращать только те строки, где значение в столбце `price` меньше 50.
