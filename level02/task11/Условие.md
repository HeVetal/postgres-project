
### Сортировка по возрастанию

В таблице `employees` содержатся информация о сотрудниках:

Напишите SQL-запрос, который выводит все записи из таблицы `employees`, отсортированные по возрасту сотрудников в порядке возрастания.

#### Требования:
1. SQL-запрос должен содержать ключевое слово `ORDER BY` для сортировки данных.
2. Сортировка должна выполняться по столбцу `age`, который содержит возраст сотрудников.
3. Сортировка должна быть выполнена в порядке возрастания (по умолчанию или с использованием ключевого слова `ASC`).
4. Результат запроса должен включать все столбцы таблицы `employees`: `employee_id`, `name`, `age`, `salary`.
