
### Выбор всех данных из таблицы

В таблице `students` содержатся данные о студентах, включая их имена, возраст и курс обучения.

Напишите SQL-запрос, который выбирает все данные из таблицы `students` с помощью команды `SELECT`.

Ожидаемый результат: запрос должен отобразить все строки и столбцы из таблицы `students`, включая их заголовки.

#### Требования:
1. Запрос должен использовать команду `SELECT *` для выбора всех столбцов из таблицы `students`.
2. Запрос должен явно указывать имя таблицы `students` после ключевого слова `FROM`.
3. Запрос должен возвращать все строки и столбцы таблицы `students`, включая заголовки столбцов.
