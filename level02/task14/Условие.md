
### Отображение топ-5 сотрудников по зарплате

В таблице `employees` содержатся записи со столбцами: `id`, `name`, `salary`.

Напишите SQL-запрос, который выбирает `name` и `salary` из таблицы `employees`, сортирует данные по зарплате в порядке убывания (`DESC`) и возвращает только 5 записей (сотрудников с самой высокой зарплатой).

#### Требования:
1. Запрос должен обращаться к таблице `employees`, содержащей столбцы `id`, `name`, `salary`.
2. Запрос должен извлекать только столбцы `name` и `salary` из таблицы `employees`.
3. Данные должны быть отсортированы по столбцу `salary` в порядке убывания (`DESC`).
4. Запрос должен возвращать только 5 записей, используя оператор `LIMIT`.
