
### Ограничение числа записей в таблице

В таблице `students` содержатся записи со столбцами `id`, `name`, `age`.

Напишите SQL-запрос, который извлекает первые 5 строк из таблицы `students`. Результат должен включать все столбцы.

#### Требования:
1. Запрос должен извлекать данные из таблицы `students`.
2. Запрос должен ограничивать количество извлекаемых строк до 5.
3. Результат запроса должен содержать все столбцы таблицы `students` — `id`, `name`, `age`.
4. Для ограничения количества строк необходимо использовать ключевое слово `LIMIT`.
