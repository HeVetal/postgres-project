
### Получить текущую дату и время

Напишите SQL-запрос, который выведет текущую дату и время.

Подсказка:

```
SELECT now();
```

#### Требования:
1. Запрос должен использовать встроенную функцию PostgreSQL `now()` для получения текущей даты и времени.
2. Запрос должен возвращать таблицу с одним столбцом, содержащим строку с текущей датой и временем.
