
### Найти текущий порт сервера PostgreSQL

Напишите SQL-запрос, который покажет текущий порт, используемый сервером PostgreSQL.

Подсказка:

```
SELECT name, setting
FROM pg_settings
WHERE name = 'port';
```

#### Требования:
1. Запрос должен использовать системное представление `pg_settings` для получения информации о настройках PostgreSQL.
2. Запрос должен включать условие `WHERE name = 'port'` для фильтрации данных и извлечения только информации о текущем порте.
3. Результат запроса должен содержать два столбца: `name` (имя настройки) и `setting` (текущее значение настройки).
