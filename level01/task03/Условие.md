
### Проверка подключения к серверу PostgreSQL с помощью SQL-запроса

Напишите SQL-запрос, который выведет версию PostgreSQL.

Подсказка:

```
SELECT version();
```

#### Требования:
1. Запрос должен использовать встроенную функцию PostgreSQL `version()` для получения информации о текущей версии PostgreSQL.
2. Запрос должен возвращать таблицу с одним столбцом, содержащим строку с информацией о версии PostgreSQL.
