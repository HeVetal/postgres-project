
### Преобразование строки к верхнему регистру

Напишите SQL-запрос, который преобразует текст 'hello world' в верхний регистр и выведет результат.

Подсказка:

```
SELECT UPPER('hello world');
```

#### Требования:
1. Запрос должен использовать встроенную функцию PostgreSQL `upper()` для преобразования строки к верхнему регистру.
2. Запрос должен возвращать таблицу с одним столбцом, содержащим строку 'HELLO WORLD'.
