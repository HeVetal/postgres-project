
### Запуск PostgreSQL-консоли

Напишите команду, с помощью которой можно подключиться к серверу PostgreSQL под пользователем postgres.

Подсказка:
```
psql -U postgres
```

#### Требования:
1. Пользователь должен подключиться к серверу PostgreSQL с помощью команды `psql`.
2. При подключении необходимо указать пользователя `postgres` с помощью ключа `-U`.
